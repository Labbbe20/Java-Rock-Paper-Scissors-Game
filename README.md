# Java-Rock-Paper-Scissors-Game

English

Java Rock Paper Scissors is a local multiplayer game developed in Java.

Two players can compete on the same computer by selecting Rock, Paper, or Scissors. After both players have made their choice, the game displays the result and highlights the winner. Visual illustrations are used to show the interaction between the selected symbols (for example, scissors cutting paper).

Features

* Local two-player gameplay
* Graphical user interface
* Visual winner and loser highlighting
* Animated result display
* Simple and intuitive controls

Deutsch

Java Rock Paper Scissors ist ein lokales Mehrspieler-Spiel, das in Java entwickelt wurde.

Zwei Spieler können an einem Computer gegeneinander antreten, indem sie Schere, Stein oder Papier auswählen. Nachdem beide Spieler ihre Auswahl getroffen haben, wird das Ergebnis angezeigt und der Gewinner hervorgehoben. Zusätzlich veranschaulichen Grafiken die jeweilige Spielsituation (z. B. schneidet die Schere das Papier).

Funktionen

* Lokaler Zwei-Spieler-Modus
* Grafische Benutzeroberfläche
* Hervorhebung von Gewinner und Verlierer
* Grafische Darstellung des Spielergebnisses
* Einfache Bedienung
